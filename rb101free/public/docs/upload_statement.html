UPLOAD prompt$; filename$

Display a widget on the web page for uploading a file, and pause to wait for the file to be uploaded.

A field will be displayed for typing in the name of a file on the local hard drive of the client user, and a browse button is also supplied so the user doesn't need to type the filename.  Once the file is specified, the user clicks on the 'Upload' button and the file is transmitted and saved into the default directory.  The name of the file is placed into filename$.  The program then continues executing.

Example:

  print "JPG uploader"
  upload "Choose a JPG file to upload. "; nameOfJpg$
  loadgraphic #jpgImage, nameOfJpg$
  render #jpgImage
