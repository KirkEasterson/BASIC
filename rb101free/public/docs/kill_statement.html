KILL filepath$

Description:

This command deletes the file specified by filepath$. The filename can include a complete path specification.

Example:

  'create a test file
  filename$ = "testfile.txt"
  open filename$ for output as #testfile
  print #testfile, "testing"
  close #testfile

  'check that the file exists
  files #dir, "testfile.txt"
  if #dir hasanswer() then print "Exists!"

  'delete the file
  kill "testfile.txt"

  'check that the file does not exist
  files #dir, "testfile.txt"
  if not(#dir hasanswer()) then print "Doesn't exist!"
