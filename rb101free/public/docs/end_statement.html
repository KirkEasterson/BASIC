Description:

This causes a program to stop executing.  Usually in web programming this isn't used since it causes all the links and other interactive objects on the page to become inactive.

Example:

  link #show3Plus4, "Add 3 to 4", [add3n4]
  print " ";
  link #sayHello, "Use your words", [speak]
  print " ";
  link #endIt, "End the program", [endIt]
  wait

[add3n4]
  print "3 + 4 = "; 3 + 4
  wait

[speak]
  print "Hello, world!"
  wait

[endIt]
  print "The program is stopped.  Try clicking a link to see what happens."
  end
