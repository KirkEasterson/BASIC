CSSID #idname, cssrule$

Description:

Sets a CSS rule that will apply to any widget that has an id matching #idname.  This can be used to change the color, size, border, etc. of widgets on a web page, or for the creation of special effects.  The CSSID statement is also useful for setting a rule for a DIV statement.

Example:

  'set a rule for objects with id "custom"
  cssid #custom, "{ background: yellow}"
  link #a, "link a", [go]
  print " ";
  link #b, "link b" ,[go]
  print " ";
  link #yel, "make me yellow", [go]
  #yel setid("custom")
  wait
