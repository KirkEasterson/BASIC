HTML expr$

Escape the contents of expr$ directly into the web page.  HTML tags will be interpreted by the browser (unlike with PRINT which always shows tags as plain text).

Example:

  html "With the HTML statement tags are interpreted, <i>for example</i>."
  print
  print "With the PRINT statement tags are not interpreted, <i>for example</i>."

Also the HTML does not insert a line break at the end.  PRINT does unless there is a trailing semicolon ';'.

See also: HEAD, PRINT
