RENDER #handle

Description:

The RENDER statement causes the object in the variable #handle to be rendered onto the web page at the current position.  The object can be one of various different kinds including Graphic, Table, Program, etc.

Some kinds of objects are rendered automatically.  Specifically these include Button, Link, Imagebutton, Textbox, Textarea, Checkbox, Radiogroup and Passwordbox

Example:

  'use the RENDER statement to display a table
  dim multiply(4, 4)
  for x = 0 to 4
    for y = 0 to 4
      multiply(x, y) = x * y
    next y
  next x
  table #tbl, multiply()
  #tbl columnnames("zero,one,two,three,four")
  #tbl cssclass("fix")
  render #tbl
