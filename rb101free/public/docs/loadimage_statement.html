LOADIMAGE nameExpr$, path$

Description:

Load an image file from the filesystem.  Locate the file using path$.  Name the loaded image using nameExpr$.

Example:

  'load an image into memory
  filepath$ = "public/cloudysky1.JPG"
  if Platform$ = "win32" then filepath$ = "public\cloudysky1.JPG"
  loadimage "sky", filepath$

  'draw the image onto a graphic object
  graphic #draw, 200, 200
  #draw drawimage("sky", 1, 1)

  'render the drawing onto the page
  render #draw

See also: GRAPHIC, LOADGRAPHIC
