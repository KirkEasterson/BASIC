EXPIRE [url$]

Description:

The EXPIRE statement forces the user's application session to expire.  An optional url$ expression can force a redirect to another page if desired.

Examples:

  link #show3Plus4, "Add 3 to 4", [add3n4]
  print " ";
  link #sayHello, "Use your words", [speak]
  print " ";
  link #endIt, "End the program", [endIt]
  wait

[add3n4]
  print "3 + 4 = "; 3 + 4
  wait

[speak]
  print "Hello, world!"
  wait

[endIt]
  expire "http://localhost:8008/sessionexpired.html"
